    ```swift
    import Foundation
    import os

    /// Per-layer TTL configuration for cached API responses.
    enum CacheLayer: String, CaseIterable {
        case earthquake
        case airQuality
        case crime
        case electricLines
        case noise
        case bundled

        /// TTL in seconds. nil means never expires.
        var ttl: TimeInterval? {
            switch self {
            case .earthquake:    return 1_800    // 30 minutes
            case .airQuality:    return 3_600    // 1 hour
            case .crime:         return 86_400   // 24 hours
            case .electricLines: return 86_400   // 24 hours (same as crime — static infrastructure)
            case .noise:         return nil       // bundled roads never expire; dynamic Overpass cached 24hr
            case .bundled:       return nil       // never expires
            }
        }
    }
    ```

    `ResponseCache` class design:
    - `static let shared = ResponseCache()` singleton
    - `private let memoryCache = NSCache<NSString, CacheEntry>()` — memory layer
    - `private let fileManager = FileManager.default` — disk layer
    - `private let diskCacheURL: URL` — computed from `FileManager.default.urls(for: .cachesDirectory, in: .userDomainMask).first!.appendingPathComponent("ResponseCache")`
    - `private let dateProvider: () -> Date` — injectable for testing, defaults to `{ Date() }`
    - `init(dateProvider: @escaping () -> Date = { Date() })` — creates disk directory if needed
    - Private `CacheEntry: NSObject` class wrapping `Data` + `Date` (stored timestamp) + `CacheLayer`

    Public API:
    - `func get(key: String, layer: CacheLayer) -> Data?` — checks memory first, then disk. Returns nil if expired or missing. Logs cache hit/miss via `AppLogger.network`.
    - `func set(data: Data, key: String, layer: CacheLayer)` — writes to both memory and disk. Disk writes the Data + metadata (stored date, layer) as a plist or JSON sidecar.
    - `func clearMemoryCache()` — calls `memoryCache.removeAllObjects()` (for testing and memory pressure)
    - `static func cacheKey(layer: CacheLayer, lat: Double, lon: Double) -> String` — quantizes lat/lon to 0.01-degree grid: `"\(layer.rawValue)_\(String(format: "%.2f", lat))_\(String(format: "%.2f", lon))"`
    - `static func cacheKey(layer: CacheLayer) -> String` — for non-location services: `"\(layer.rawValue)_global"`

    Disk storage format: For each cache key, write two files in the `ResponseCache/` directory:
    - `{key}.data` — the raw response Data
    - `{key}.meta` — JSON `{"storedAt": timestamp, "layer": "earthquake"}`

    Expiry check: `let elapsed = dateProvider().timeIntervalSince(storedDate); if let ttl = layer.ttl, elapsed > ttl { return nil }`

    Log pattern: `AppLogger.network.info("Cache HIT [\(layer.rawValue)] key=\(key)")` and `AppLogger.network.info("Cache MISS [\(layer.rawValue)] key=\(key)")`.

    **REFACTOR phase:** Ensure `memoryCache.countLimit = 50` to prevent unbounded memory growth. Ensure disk directory creation is guarded with `try?`.
