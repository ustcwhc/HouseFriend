    Both services fetch a fixed Bay Area bounding box (no location params), so they use the simple `ResponseCache.cacheKey(layer:)` global key.

    **EarthquakeService.swift — modify `fetch()` method:**

    At the top of `fetch()`, before the URLSession call, add cache check:
    ```swift
    let cacheKey = ResponseCache.cacheKey(layer: .earthquake)
    if let cachedData = ResponseCache.shared.get(key: cacheKey, layer: .earthquake) {
        // Decode from cached data — reuse existing decode logic
        do {
            let decoded = try JSONDecoder().decode(USGSResponse.self, from: cachedData)
            let events = decoded.features.compactMap { ... } // same compactMap as existing
            AppLogger.network.info("Earthquake: loaded \(events.count) events from cache")
            DispatchQueue.main.async {
                self.events = events
                self.isLoading = false
            }
            return
        } catch {
            // Cache data corrupted, fall through to network
            AppLogger.network.warning("Earthquake: cache decode failed, fetching from network")
        }
    }
    ```

    After successful network fetch and decode (after the `let events = ...` line, before the `DispatchQueue.main.async` that sets `self?.events`), add cache write:
    ```swift
    ResponseCache.shared.set(data: data, key: cacheKey, layer: .earthquake)
    ```

    Extract the `compactMap` closure into a private static method `private static func parseEvents(from decoded: USGSResponse) -> [EarthquakeEvent]` to avoid duplicating the decode logic between cache and network paths.

    **ElectricLinesService.swift — modify `fetch()` method:**

If hit, decode the cached GeoJSON data using the existing `JSONDecoder().decode(GeoJSONFeatureCollection.self, from:)` path. If miss, proceed to URLSession. After successful network fetch, call `ResponseCache.shared.set(data: data, key: cacheKey, layer: .electricLines)`.

    Extract the line-parsing logic into a private static method to reuse between cache hit and network fetch paths.

    **Important:** Do NOT change any `@Published` property types, function signatures, or public API. Only modify the internal fetch flow.

    At the top of `fetch(lat:lon:)`, before the URLSession call:
    ```swift

    if let cachedData = ResponseCache.shared.get(key: cacheKey, layer: .airQuality) {
        if let json = try? JSONSerialization.jsonObject(with: cachedData) as? [String: Any],
           let current = json["current"] as? [String: Any],
           let aqi = current["us_aqi"] as? Int {
            DispatchQueue.main.async {
                self.data = AirQualityData(
                    aqi: aqi,
                    category: Self.categoryForAQI(aqi),
                    pollutant: "PM2.5"

                self.isLoading = false
            }
            return
        }
    }
    ```

    ```swift
    ResponseCache.shared.set(data: data, key: cacheKey, layer: .airQuality)
    ```
