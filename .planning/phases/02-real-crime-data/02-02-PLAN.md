    1. Add a thread-safe density grid property:
       ```swift
       private let gridLock = NSLock()
       private var _densityGrid: DensityGrid?
       var densityGrid: DensityGrid? {
           get { gridLock.lock(); defer { gridLock.unlock() }; return _densityGrid }
           set { gridLock.lock(); defer { gridLock.unlock() }; _densityGrid = newValue }
       }
       ```
       Thread safety is critical because `loadTile()` runs on background threads while the grid is set from the main thread (per Pitfall 7 in RESEARCH.md).

    2. Replace `static func crimeValue(lat:lon:)` with an instance method:
       ```swift
       func crimeValue(lat: Double, lon: Double) -> Double {
           guard let grid = densityGrid else { return 0.0 }  // No data = transparent (not fake heatmap)
           let raw = grid.intensity(lat: lat, lon: lon)
           guard raw.isFinite else { return 0.0 }
           return max(0.10, min(1.0, 0.10 + raw * 0.90))
       }
       ```
       When `densityGrid` is nil, return 0.0 (transparent) — this means areas with no data show NO heatmap, addressing the "no fake data" requirement. The 0.10 base ensures non-zero cells are always visible.

           // Numbered circle: white background, colored border by severity
