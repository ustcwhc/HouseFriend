    **Step 3: Migrate ZoomTier to support Mapbox zoom levels.**
    Add a new initializer `init(zoom: Double)` to ZoomTier that maps Mapbox zoom levels (0-22 scale) to the existing tiers. The mapping is:
    - satellite: zoom < 5 (corresponds to span > 5.0 deg)
    - state: zoom 5-8 (corresponds to span 1.2-5.0 deg)
    - county: zoom 8-11 (corresponds to span 0.3-1.2 deg)
    - city: zoom 11-14 (corresponds to span 0.08-0.3 deg)
    - neighborhood: zoom >= 14 (corresponds to span < 0.08 deg)

    Keep the existing `init(span:)` and `init(region:)` for backward compatibility (other files still use MapKit). Add:
    ```swift
    /// Determine zoom tier from Mapbox camera zoom level (0-22 scale).
    init(zoom: Double) {
        switch zoom {
        case _ where zoom >= 14: self = .neighborhood
        case _ where zoom >= 11: self = .city
        case _ where zoom >= 8:  self = .county
        case _ where zoom >= 5:  self = .state
        default:                 self = .satellite
        }
    }
    ```
    Import `MapKit` remains (other code uses it). No need to add `import MapboxMaps` to ZoomTier.
