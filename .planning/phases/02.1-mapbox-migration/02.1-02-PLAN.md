<objective>
Migrate all polygon and polyline overlay layers from MapKit MKPolygon/MKPolyline to Mapbox GeoJSON sources with FillLayer and LineLayer. This covers: fire hazard zones, electric lines, odor zones, ZIP code polygons (with labels), and noise road smoke effect.

Purpose: These 5 overlay-based layers represent the bulk of the map's visual data. Migrating them to Mapbox's layer system replaces ~200 lines of MapKit overlay/renderer code with GPU-accelerated vector layers.
Output: All polygon/polyline layers render correctly through Mapbox GeoJSON sources and layers.
</objective>
