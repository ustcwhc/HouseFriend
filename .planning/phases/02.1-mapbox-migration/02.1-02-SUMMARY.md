# Phase 02.1 Plan 02: Overlay Layers Summary

**5 polygon/polyline overlay layers migrated from MapKit MKOverlay/MKRenderer to Mapbox GeoJSON sources with FillLayer, LineLayer, SymbolLayer, and 4-layer smoke blur effect**

## Performance

- **Duration:** 12 min

## Accomplishments

- Fire hazard zones render as severity-colored FillLayer polygons (dark red/orange-red/orange/yellow via match expression)
- Electric lines render as yellow LineLayer polylines at 2.5pt width
- Odor zones render as brown FillLayer with brown LineLayer stroke
- 445 ZIP code polygons render with highlight expression (selected=pink, default=gold) and SymbolLayer text labels at centroids
- Noise roads render with 4-layer smoke/haze effect: outer haze (12x width, blur 8), mid haze (6x, blur 4), inner smoke (3x, blur 2), core colored line (1x, no blur)
- Railway noise segments render with dashed line pattern via separate filtered LineLayer
- All overlay layers use declarative MapContent builder pattern matching existing crime heatmap architecture

## Files Created/Modified

- `HouseFriend/Views/HFMapView.swift` - Added 5 overlay layer content builders (fireLayerContent, electricLayerContent, odorLayerContent, zipLayerContent, noiseLayerContent), 6 GeoJSON helper properties, 3 noise helper functions

## Decisions Made

- Used rgba string literals for colors in match/step expressions because `StyleColor(red:green:blue:alpha:)` returns optional which doesn't conform to `ExpressionArgumentConvertible`
- Created helper functions (`makeNoiseCoreLayer`, `makeNoiseRailwayLayer`) for layers needing `filter` property since it's not available as a chainable builder method
- Separate GeoJSONSource for ZIP labels because SymbolLayer needs Point geometry while polygon source has Polygon geometry
- Omitted `GeoJSONSource.tolerance` setter since no builder method exists; default (0.375) is adequate for 445 ZIP polygons

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] MapContentBuilder/MapContent type ambiguity**
- **Found during:** Task 1
- **Issue:** `MapContentBuilder` and `MapContent` ambiguous between MapboxMaps and _MapKit_SwiftUI when MapKit is imported
- **Fix:** Fully qualified as `MapboxMaps.MapContentBuilder` and `MapboxMaps.MapContent` on all computed properties
- **Files modified:** HouseFriend/Views/HFMapView.swift
- **Verification:** Build succeeds
- **Committed in:** c245df1

**2. [Rule 3 - Blocking] StyleColor optional return in expressions**
- **Found during:** Task 1
- **Issue:** `StyleColor(red:green:blue:alpha:)` returns optional, failing ExpressionArgumentConvertible conformance in match expressions
- **Fix:** Switched to rgba string format (e.g., `"rgba(184,5,5,1.0)"`) matching crime heatmap pattern
- **Files modified:** HouseFriend/Views/HFMapView.swift
- **Verification:** Build succeeds
- **Committed in:** c245df1

**3. [Rule 3 - Blocking] .constant()/.expression() type inference failure**
- **Found during:** Task 1
- **Issue:** `.constant()` and `.expression()` calls lost type context when return type became `some MapboxMaps.MapContent`
- **Fix:** Switched to direct builder methods (e.g., `.heatmapOpacity(0.85)` instead of `.heatmapOpacity(.constant(0.85))`)
- **Files modified:** HouseFriend/Views/HFMapView.swift (crime layer code)
- **Verification:** Build succeeds
- **Committed in:** c245df1

**4. [Rule 3 - Blocking] MapContentBuilder var pattern incompatible with filter**
- **Found during:** Task 3
- **Issue:** `var layer = LineLayer(...); layer.filter = ...; layer` pattern fails with `@MapContentBuilder` — can't return modified struct as MapContent
- **Fix:** Extracted into helper functions returning `LineLayer` (works because LineLayer conforms to MapContent)
- **Files modified:** HouseFriend/Views/HFMapView.swift
- **Verification:** Build succeeds
- **Committed in:** 6338340

**5. [Rule 3 - Blocking] GeoJSONSource.tolerance not available as builder method**
- **Found during:** Task 2
- **Issue:** Plan specified `.tolerance(0.0005)` but GeoJSONSource only has builder methods for `data`, `prefetchZoomDelta`, `tileCacheBudget`
- **Fix:** Omitted tolerance — default (0.375) is adequate for 445 ZIP polygons
- **Files modified:** N/A (omitted rather than added)
- **Verification:** Build succeeds
- **Committed in:** 6338340
