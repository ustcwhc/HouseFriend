## Files Created/Modified

## Decisions Made

- Used `@_spi(Experimental) import MapboxMaps` because all source types (`GeoJSONSource`, `VectorSource`, etc.) require this SPI attribute for declarative `MapStyleContent` conformance in v11
- Replaced `StyleColor` objects in Mapbox expressions with `rgba()` CSS strings because `StyleColor` does not conform to `ExpressionArgumentConvertible`
- Used `MapViewAnnotation` (SwiftUI view annotations) instead of `PointAnnotation` (image-based) because it supports SF Symbols natively without pre-registration
- Tracked zoom level via `@State private var currentZoom` updated in `onCameraChanged` for annotation visibility

- **Found during:** Task 1 (HeatmapLayer color gradient)

- **Fix:** Replaced all `StyleColor(red:green:blue:alpha:)` in expressions with `"rgba(r,g,b,a)"` CSS color strings
- **Files modified:** HouseFriend/Views/HFMapView.swift
